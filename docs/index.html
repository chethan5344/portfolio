Latest Project

LiveOrder

LiveOrder is a B2B platform that streamlines medical retail and distribution by connecting distributors and retailers. It enables efficient stock management, sales tracking, and order processing through a role-based system tailored for sellers, buyers, and administrators.

Key Contributions:

  • Implemented role-based access for sellers, buyers, and admins, ensuring secure and seamless user experiences.
  • Developed sales reports and analytics dashboards for better decision-making.
  • Built stock management features to help distributors monitor and control inventory in real time.
  • Designed and integrated onboarding workflows for new clients, improving user engagement.
  • Rewrote and stabilized the payment module, which was previously causing frequent bugs, delivering a permanent and reliable fix.
  • Optimized UI components for better performance, accessibility, and responsiveness across devices.
  • Collaborated with backend teams to ensure smooth API integrations and data handling.
  • Revamped 80% of the front-end UI based on new designs, improving user experience and maintaining design consistency across modules.
  • Minimized the overall front-end codebase by 30% through refactoring and removal of redundant logic.

Impact:

  • Enhanced the efficiency of medical retailers and distributors in managing their business.
  • Improved user engagement with a more intuitive and scalable frontend.
  • Reduced order processing time by optimizing workflows and implementing robust features.

Technologies used include:

  • React
  • Javascript
  • Redux
  • React Router
  • Webpack
  • Docker
